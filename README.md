# ระบบลงเวลาเรียนด้วย AI (Attendance System)

ระบบเช็คชื่อด้วย AI ตรวจสอบใบหน้าและ GPS พร้อมแจ้งเตือนผ่าน Telegram

## 🚀 คุณสมบัติ

- ✅ ลงเวลาด้วย AI ตรวจสอบใบหน้า
- ✅ Liveness Detection (ป้องกันการปลอมแปลง)
- ✅ ตรวจสอบตำแหน่งด้วย GPS
- ✅ แจ้งเตือนผ่าน Telegram
- ✅ PWA รองรับมือถือและคอมพิวเตอร์
- ✅ Firebase Authentication
- ✅ ระบบรายงานและสถิติ

## 📦 การติดตั้ง

```bash
# Clone project
git clone <your-repo>
cd attendance-system

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Deploy to Firebase
npm run deploy
